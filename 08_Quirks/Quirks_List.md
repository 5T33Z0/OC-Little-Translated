- `XhciPortLimit`: Disable for macOS 11.3 and newer â€“ create a custom USB Port Map instead!

- AMD CPUs also require additional [**Kernel patches**](https://github.com/AMD-OSX/AMD_Vanilla/tree/master) to run macOS.
