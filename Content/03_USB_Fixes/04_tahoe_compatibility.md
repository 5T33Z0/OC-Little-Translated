# Adjusting USBMap Kexts for macOS Tahoe Compatibility

In order for USBMap kexts to work in macOS Tahoe, the `info.plist` inside the kext needs to be adjusted.

## Instructions

For detailed instructions on making your USB mapping kexts compatible with macOS Tahoe, please refer to:

→ [**macOS Tahoe USB Compatibility Guide**](https://github.com/5T33Z0/OCLP4Hackintosh/blob/main/Enable_Features/USB_Tahoe.md)

## What You'll Learn

- How to modify the `info.plist` file within USB mapping kexts
- Specific changes required for macOS Tahoe compatibility
- How to verify that your modifications are working correctly

## Important Notes

> [!NOTE]
> 
> - This applies to kexts generated by USBMap, Hackintool, and similar tools
> - The modifications are necessary due to changes in how macOS Tahoe handles USB port definitions
> - Always keep a backup of your working EFI before making changes

## Compatibility

This guide covers compatibility adjustments for:
- `USBPorts.kext`
- `UTBMap.kext`
- Other custom USB mapping kexts
